# Utility Skills

Collection of some utility skills.

I created these for my personal use cases, so there will be some assumptions, requirements and specific paths and structures in the skill body you will need to edit to make it your own:

- I use these within an [Obsidian](https://obsidian.md) Vault.  My vault is structured using [PARA](https://fortelabs.com/blog/para/), expect to see some references to that.
- I assume that [Pandoc](https://pandoc.org/) is installed.
- I assume that you have (at the time of writing this) a [Catalyst license](https://help.obsidian.md/catalyst) for Obsidian, and that you have the [Obdidian CLI](https://help.obsidian.md/cli) installed.
